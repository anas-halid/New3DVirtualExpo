Deploy Locally :
<br>
<b>Backend:</b>
Step 1: Open terminal
Step 2: Direct your file to "file" [cd file]
Step 3: Direct your file to "server" [cd server]
Step 4: Start Server [node index]
<br>
<b>Frontend:</b>
Step 1: Open new terminal
Step 2: Direct your file to "file" [cd file]
Step 3: Start React Frontend [npm start]
